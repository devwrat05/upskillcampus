# File Organizer using Python

## Overview

This project was developed as part of the Python Programming Internship conducted by upskill Campus in collaboration with UniConverge Technologies Pvt. Ltd.

The File Organizer automatically organizes files into different folders based on their file extensions. It helps keep folders clean and reduces the manual effort of sorting files.

---

## Features

- Organizes files automatically
- Creates folders if they do not exist
- Supports multiple file types
- Easy to understand and modify
- Beginner-friendly Python project

---

## Technologies Used

- Python 3
- os module
- shutil module

---

## Supported File Types

| File Type | Folder |
|-----------|---------|
| .jpg, .jpeg, .png | Images |
| .mp4 | Videos |
| .pdf, .docx | Documents |
| .mp3 | Audio |
| .exe, .msi | Applications |
| .zip, .py | Other Files |

---

## How to Run

1. Install Python 3.
2. Update the folder path in the Python file.
3. Run the program.
4. The files will be organized automatically into their respective folders.
